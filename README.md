# Оно Тебе Надо

Аукцион вещей в которые никто не верил...

# Описание

Верстка выполнена по макету с отклонениями от рекомендаций из теории. Так, grid-layout использован только для карточек, поскольку они распределены равномерно. В шапке использована flexbox-верстка для более удобного центрирования логотипа. Разбивка на классы отличается за счет выделения общих классов (что привело к уходу от БЭМ, поскольку элементы меню утратили специфичность).
